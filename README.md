# express-react-node

In `backend` run:

```npm i
cd backend
npm run awesome
```

In `frontend` run:

```
npm i
cd frontend
npm run serve
```

Go to your localhost
`http://localhost:3000/`


Follow this medium article for more instruction!
[Medium Article](https://medium.com/@moises.trejo0/express-react-mui-typescript-heroku-app-9f85cda54451)
